# MIPS32 Processor

A simple 5-stage pipelined MIPS32 processor implemented in Verilog.
Pipelined implementation of subset of MIPS32 instruction set.

## Features
- 32-bit architecture
- Five-stage pipeline
- Register file
- ALU
- Instruction and data memory
